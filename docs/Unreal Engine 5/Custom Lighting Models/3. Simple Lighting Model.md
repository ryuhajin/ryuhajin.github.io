# 3. Simple Lighting Model
기본적인 Ambient, Diffuse, Specular 조명을 결합한 라이팅 셰이더 만들기

> 앞에서 만들었던 준비물에 roughness, AO, Normal map 등을 섞기
- Ambient (주변광) : 가장 단순한 형태의 주변광 color 3개 lerp
- Diffuse (확산광) : Lambert diffuse 모델
- Specular (반사광) : Blinn-Phong 모델

---

## Simple Lighting Material Function 만들기

### input

| 이름	 | 타입	 | 기본값	 | 설명 |
|---|---|---|---|
| NormalWS	 | Vector 3	 | VertexNormalWS 노드  |월드 공간 노멀  |
| BaseColor	 | Vector 3	 | 0.5, 0.5, 0.5 (회색)  |	텍스처 색상  |
| Roughness	 | Scalar	 | 0.5	 | 거칠기 (0=매끄러움, 1=거침)  |
| AmbientOcclusion  |	Scalar  |	1.0	 | 앰비언트 오클루전 (1=없음, 0=완전 차폐)  |

## 전체 라이팅 연산 과정
> 기초 배경광 -> 난반사광(형태) -> 정반사광(광택) 순서로 진행

```md
1. **배경 설정 (Ambient Step)** : `Ambient Color * AO`
    - 가장 밑바탕이 되는 환경 빛을 먼저 정의하고 틈새를 가림
2. **형태 부여 (Diffuse Step)** : `Base Color * Lambert Diffuse + Ambient`
    - 물체의 고유 색상(Albedo)은 빛의 산란과 환경광에 영향을 받음
3. **하이라이트 추가 (Specular Step)** : `Specular Result * Fresnel`
    - 스펙큘러는 표면에서 튕겨 나오는 "빛의 반사"이므로, 물체 고유의 색(Base Color)과는 별개로 그 위에 덧씌워짐
4. **최종 출력 (Final Step)** : `Diffuse (2번 결과) + Specular (3번 결과)`
    - 빛은 가산(Additive) 혼합이기 때문에 더함
    - 모든 빛의 에너지를 합쳐서 최종 픽셀 색상을 결정
```

# 사용 노드
## TransformVector (공간 변환)
벡터를 한 좌표계(Space)에서 **다른 좌표계로 변환**

- 노멀 맵 텍스처 데이터는 Tangent Space에 저장
- **수학적 연산은 동일한 공간에서 이루어져야 하므로**, 노멀 벡터를 월드 공간으로 옮겨주는 과정이 필수

---

## Exp2 (지수 연산 - Base 2)
밑이 2인 지수 함수: **출력 = 2^(입력)**

- 0~1 사이의 선형적인 값을 지수적인 큰 값 (예: 2, 4, 8 ... 1024)으로 뻥튀기하여, 하이라이트의 날카로움(Power)을 조절하기 위해 사용

---

## Fresnel (프레넬 효과)
**표면 노멀과 카메라 방향 사이의 각도에 따른 반사율 차이를 계산**

- 모든 사물은 정면으로 볼 때보다 비스듬하게 볼 때 더 반사가 잘일어남
- 해당 물리적 현상을 구현하여 스펙큘러의 강도를 조절

1. **F_0 (기본 반사율)** : 물체를 **정면(0도)에서 바라봤을 때의 반사율**
    - 비금속은 약 0.04, 금속은 Base Color
2. **F_90** : 물체의 **외곽(90도, Glancing Angle)을 바라봤을 때의 반사율**
    - 거의 모든 사물은 여기서 100% (1.0)의 반사율을 갖는다

```
Fresnel은 바라보는 각도에 따라 F_0에서 F_90 사이를 어떻게 부드럽게 연결할지 결정하는 수학적 공식
```

## 1. Normal map 적용
### 노멀 맵 공간 변환
노멀 맵에서 읽어온 데이터는 Tangent Space(접선 공간) 기준

> 라이팅 계산 시, 원래 모델의 기하학적 법선 (버텍스) 대신 이 텍스처에서 읽은 법선을 사용

1. Normal Map 텍스처에서 RGB 값을 읽음 (보통 Tangent Space 기준)
2. Tangent Space → World Space 변환 (또는 엔진에 따라 다른 공간)
3. 변환된 법선을 사용하여 Diffuse, Specular 계산:
    - Diffuse: `max(0, dot(normal, lightDir))`
    - Specular: `dot(normal, halfDir)` 등을 power 연산

---

## 2. Roughness 개념 및 Specular와의 관계
### Roughness vs Specular
- Roughness : 0 → 매우 매끄러움 (거칠기 없음)
- Specular : Power가 작을수록 하이라이트가 크고 흐릿 (거친 표면)

> 따라서 Roughness 값과 Power 값은 반비례 관계

```c++
Smoothness = 1 - Roughness   //(Roughness 0~1 입력)
Linear = Smoothness * 10 + 1   //(범위: 1 ~ 11)
ExpValue = 2 ^ Linear            //(지수 결과: 2^1 ~ 2^11 = 2 ~ 2048)
Power = ExpValue
```
