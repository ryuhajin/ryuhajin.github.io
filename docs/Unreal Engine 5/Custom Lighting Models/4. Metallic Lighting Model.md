# 4. Metallic Lighting Model
앞서 만든 simple lighting에 metallic 파라미터 추가하기
- 금속(Metal)은 일반적인 사물과는 빛을 반사하는 방식이 다름

## 목표
- metallic 파라미터를 추가해서 비금속 / 금속 두 라이팅 모델을 하나의 셰이더 안에서 전환

---

# 사용 노드
## Texture Sample : Mip Value Mode (밉 값 모드)
밉맵(Mipmap)은 텍스처의 여러 해상도 레벨을 미리 저장해두는 기법
- 거리가 멀거나 표면이 거칠 때 낮은 해상도를 사용하여 성능과 품질을 최적화

| 모드	| 설명 |
|---|---|
| None	| 표준 자동 모드. 카메라 거리와 픽셀 밀도(DDX/DDY)를 기반으로 GPU가 가장 적합한 밉 레벨을 자동으로 선택 |
| Mip Level (Absolute)	| 특정 MIP 레벨을 직접 지정 (0 = 최대 해상도, 값이 클수록 흐릿) |
| Mip Level (Bias)	| 오프셋 모드. 자동 레벨에 사용자가 지정한 값 더하기 (양수면 더 흐릿, 음수면 더 선명) |
| Derivative (Auto)	| 수동 도함수 모드. 자동 계산이 아니라 사용자가 직접 벡터(DDX, DDY)를 입력하여 밉 레벨을 결정 |

```
Roughness를 이용해 reflection cubemap의 MIP level을 직접 지정
- Roughness가 낮을수록: 낮은 MIP 값(0 근처) → 선명한 반사
- Roughness가 높을수록: 높은 MIP 값(예: 8) → 흐린 반사
```

---

# 금속의 물리적 특성
## 1. Diffuse의 부재
  - 순수 금속은 빛을 내부로 흡수하여 산란시키지 않음.
  - 즉, 난반사(Diffuse)가 0
  - 모든 빛은 표면에서 정반사

> 따라서 diffuse과 곱할 색상을 metallic을 알파로 넣은 lerp 노드로 선택한다

### Lerp(Base Color, 0.0, Metallic) * diffuse
metallic이 1이면 `diffuse * 0` = diffuse 없음

```c++
diffuseColorFactor = lerp(BaseColor, 0.0, Metallic)

directDiffuse = diffuse * diffuseColorFactor
ambientDiffuse = ambientDiffuse * diffuseColorFactor
```

---

## 2. Tinted Specular (유색 하이라이트)
  - 비금속의 하이라이트는 광원의 색을 따라감 (보통 흰색)
  - 하지만 금속은 **금속 고유의 색상(Base Color)**이 하이라이트에 묻음

## 3. 높은 반사율
    - 비금속의 정면 반사율이 4% 내외인 것에 비해, 금속은 매우 높은 반사율을 가짐
    - 금속의 Ambient Specular는 환경 자체가 반사가 됨

> 비금속일 때는 기존 reflectance 값(Fresnel 기반, 정면 약 4%) 사용

> 금속일 때는 BaseColor를 reflectance 값으로 사용

### Lerp(Fresnel, Base Color, Metallic)
```
metallic = 1이면 specular reflectance로 BaseColor를 사용
    - 즉, direct specular와 ambient specular가 BaseColor에 의해 색이 입혀짐

반면 diffuse 계열은 0 
```

---

## Ambient Specular 추가하기
Ambient Specular은 reflection vector로 cubemap을 샘플링해 만든다

> roughness를 이용해 MIP level을 0~8 범위로 lerp하여, 거칠수록 더 높은 MIP(더 흐린 반사)를 샘플링

### Lerp(0.0, 8.0, Roughness)
- roughness = 0 : mipmap 값 = 0, 선명한 환경 반사광 (Ambient Specular) 생성
- roughness = 1 : mipmap 값 = 8, blur 처리된 환경 반사광 생성

---

## 라이팅 결합 flow chart

```
[Diffuse Color Factor]
Lerp(BaseColor, 0.0, Metallic)
    ├─► Direct Diffuse   = DirectDiffuseTerm   * DiffuseColorFactor
    └─► Ambient Diffuse  = AmbientDiffuseTerm  * DiffuseColorFactor

[Reflectance Factor]
Lerp(NonMetalReflectance(Fresnel 기반), BaseColor, Metallic)
    ├─► Direct Specular  = DirectSpecularTerm  * ReflectanceFactor
    └─► Ambient Specular = ReflectionCubemap   * ReflectanceFactor

Final Result
= (Direct Diffuse + Direct Specular) + (Ambient Diffuse + Ambient Specular)
```
