# 4. Fullscreen Masks
세 가지 유형의 전체 화면 마스크 만들기

---

# 그래프 예제

## Square mask
화면 가장자리에 검은 사각 테두리 생성

1. **texcoord * scalar(4)**
    - 스케일 4배로 늘리기
    - **대각선 위 마스크 생성**
2. **1-x(texcoord) * scalar(4)**
    - **대각선 아래 테두리 생성**
3. **1,2번 각각 saturate 한 뒤 곱하기**
4. **component mask로 R,G 뽑기**
    - **흑백 마스크 생성을 위해**
5. **뽑은 r,g 채널 곱한 뒤 saturate**
6. **SceneTexture:PostProcessInput0과 곱하기**
    - 화면에 마스크 테두리 생성

> scalar 값이 작을수록 마스크 테두리 커짐

---

## Rounded Mask
화면 가장자리에 CRT 모니터 같은 둥근 모서리 효과

1. **texcoord * 1-x(texcoord)**
    - 화면 중앙 기준 시메트리 같이 섞임
2. component mask로 R,G 뽑기
    - **흑백 마스크 생성을 위해**
3. **C = R * G**
4. **C * C**
    - power(C, 2) 와 같은 효과
5. **scalar(8000) * power**
    - scalar 값으로 전체적으로 어두워진 부분 밝기를 올려줌
6. **saturate 후 SceneTexture:PostProcessInput0과 곱하기**
    - 화면에 마스크 테두리 생성

> scalar 값이 작을수록 마스크 테두리 커짐

---

## Distance Mask
비네트 효과 (가장자리 어두움)를 위한 거리 기반 마스크

1. **texcoord - V2(0.5, 0.5)**
    - 화면 중앙으로 좌표 옮기기
2. **T = scalar(0.6) * 1번 결과 값**
    - 스칼라 값이 작으면 흐린 큰 원 생성 
3. **vector Length(T)**
    - 거리 벡터 얻기
4. **saturate(vector Length)**
5. **power(length, 1.5)**
    - 가운데는 어둡고 가장자리는 밝은 마스크 생성
6. **1-x 노드로 반전한 뒤 밝기 스칼라 add**
    - 가운데는 밝고 가장자리는 어두운 마스크로 만들기
    - 그 뒤 add(0.02) 와 같은 값으로 전체적인 밝기 올려줌
7. **saturate 후 SceneTexture:PostProcessInput0과 곱하기**
    - 화면에 마스크 테두리 생성

> scalar 값이 클수록 원이 작아지고 선명해짐
