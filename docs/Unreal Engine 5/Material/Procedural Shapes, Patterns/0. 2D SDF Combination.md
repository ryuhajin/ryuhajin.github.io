# 0. 2D SDF C
**두 도형의 SDF를 d1, d2라 할 때, 조합 함수 combine(d1, d2)를 정의하면 복합 도형의 SDF를 만들 수 있다**

## 조합 종류
1. **불리언(Boolean) 조합 (합집합/교집합/차집합)**
2. **연속적 조합 (보간)**
3. **접합부**를 미적으로 바꾸는 조합 **(라운드/챔퍼/경계 효과)**
