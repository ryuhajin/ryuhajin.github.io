# 1. Introduction To HLSL
노드 기반 머티리얼 생성 후 HLSL 코드로 변환하기 위해 역공학하는 과정 설명

---

## 그래프 예제
좌우로 움직이는 슬라이드 애니메이션 마스크

## 노드 기반
1. Texture Coordinate: UV 좌표 얻기
2. Multiply: 그리드 크기 조정 (×10)
3. Ceil: 가장 가까운 정수로 반올림 (블록 효과 생성)
4. Divide: 값 범위를 [0,1]로 정규화 (÷10)
5. Sine(Time): 웨이브 애니메이션 생성
6. Subtract(ceil(uv), sin(time)): 그리드에서 웨이브 값 빼기
7. Step(0.5): 부드러운 그라데이션을 단색 마스크로 변환
8. Lerp: 마스크를 사용한 두 색상/텍스처 블렌딩

## 코드 기반
노드 → 코드 변환 과정

1. Step 노드: step(Y coord, 0.5)
2. Subtract 노드: (A - B)
3. Divide 노드: (A / 10)
4. Ceil 노드: ceil(UV × 10)
5. Sine 노드: sin(Time)

### 최종 HLSL 코드

```c++
return step((ceil(UV * 10) * 0.1) - (sin(time)), 0.5);
```
