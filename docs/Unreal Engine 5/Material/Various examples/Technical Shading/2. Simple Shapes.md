# 2. Simple Shapes
노드, HLSL을 사용해 원 그리기

---

## 원의 방정식

```
기본 공식: x² + y² = r²

중심점 이동 공식: (x-h)² + (y-k)² = r²
```

---

# 노드 기반
## x^2 + y^2 = r^2
1. `tex coord` - V2(0.5, 0.5)
    - 좌표를 중심으로 이동
2. `breakoutFloat2` 로 각각 x,y 분리
3. 분리된 x * x, y * y 로 제곱
4. 3번 결과 더하기

## vector length
1. `tex coord` - V2(0.5, 0.5)
    - 좌표를 중심으로 이동
2. `Vector Length` V2에 연결하여 길이 구하기
3. `Power(length, sine(time))` 으로 원이 커졌다 줄어들었다 하는 애니메이션 출력
    - 지수(exp) : 원 size

---

# HLSL
---

## input
1. UV (texCoord)
2. tile : 몇번 반복할것인지 (grid)
    - `frac(uv * size)`
3. Position (float2) : 원 중심점 좌표
4. radius : 중심까지의 거리 (그라데이션을 0,1로 명확하게 마스킹)
    - `step(radius)` = `d <= radius`

### 최종 HLSL 코드

```c++
float d = length(frac(uv*tile) - pos);

return (d<=radius);
```

- radius에 sine(time)으로 커졌다 작아졌다 애니메이션 가능
