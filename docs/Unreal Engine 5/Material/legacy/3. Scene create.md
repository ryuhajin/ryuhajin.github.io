## 4. decal
- 표면에 붙일 수 있는 스티커
- 메시에 투사시킬수 있는 머티리얼
- sort order : 레이어 순서 (데칼이 겹쳤을 때 어느것이 위인지 결정)
- wet decal : 데칼이 웅덩이에 영향을 받게 하고싶으면 마스터 머티리얼 복사해서 wet_decal 만든 뒤 러프니스 연결 끊기
- 스태틱 메시가 데칼의 영향을 받지 않게끔 하려면 Advanced의 Receives Decals 옵션 끄기
