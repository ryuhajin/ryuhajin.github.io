### Blend mode for VFX
1. translucent
    - 어두운 컬러감 사용 가능
2. additive
    - 검은색이 투명하게 렌더링
    - 밝음만 표현할 때 사용

### shading mode for VFX
1. default lit
    - PBR에서 사용되는 기본
    - 그림자와 빛 사용
2. unlit
    - vfx 머티리얼에는 unlit 사용
    - 그림자나 빛이 드리우지 않음
