# 3. Vector node
벡터에 관한 노드들

---

### get Actor ~ vector
1. get actor forward vector
    - 오브젝트 x축 가져오기
2. get actor right vector
    - 오브젝트 y축 가져오기
3. get actor up vector
    - 오브젝트 z축 가져오기

> unit 벡터 (크기가 1) 반환함

---

### Break Vector
벡터값 분리

1. get actor location 의 리턴 값을 분리하고 싶을 때
2. `Break Vector` 노드로 **x,y,z 벡터 값 분리**
3. 원하는 축의 값에 더하기, 빼기등 작업 수행

---

### Make Vector
벡터 만들기. 분리된 벡터 합치기

- **새 벡터를 만들거나 분리된 벡터를 합칠 때** 사용

---

### vector length
**벡터 길이 (크기) 구하기**

---

### normalize
입력된 벡터와 **방향은 같고 크기가 1인 벡터 출력**

---

### get Unit Direction (vector)
**두 지점 사이의 방향 얻기**

1. from (location)
2. to (location)
3. 출력 : **from -> to로 향하는 방향 단위 벡터**

> A에서 B로 가는 벡터를 찾고 싶으면 B - A

---

### Find look at Rotation
시작 위치가 목표 위치를 바라보는데 필요한 회전 출력

- `look at`은 객체의 전방 벡터를 의미함
