# Shader Editor
블렌더 기초 쉐이더 에디터 알아보기

## 머티리얼 생성
1. 매쉬 선택된 상태에서
2. 오른쪽 패널에서 빨간 공 아이콘(Material Properties) 클릭
3. New 버튼 클릭

> 새 머티리얼이 생성되고 선택된 매쉬에 자동으로 적용됨

- 블렌더 **기본 뷰포트 모드는 머티리얼이 안 보이는 모드**
- 3D 뷰포트 오른쪽 위에 있는 구 아이콘들 중
    - Material Preview (체커무늬 구) or Rendered 클릭
    - **머티리얼 색상 보임**

---

## Shader Editor의 기본 구성
Shader Editor를 열면 항상 최소 2개의 노드가 존재

```md
[ Principled BSDF ] ───▶ [ Material Output ]
```

1. **Material Output (출력)**
    - 머티리얼의 최종 결과
2. **Principled BSDF (표준 표면 셰이더)**
    - 현실 물리 기반(PBR)

---

### 절차적 텍스쳐 생성 구조

```md
[ Noise Texture ] → [ Color / 수치 ] → [ Principled BSDF ] → [ Material Output ]
```

- **Noise Texture** = 패턴/값 생성기
- **Principled BSDF** = 표면 물리 계산기
- **Output** = 화면 출력

---

## 소켓 색상

소켓 색상 | 데이터 타입 (Data Type) | 설명 및 특징 |
|---|---|---|
노란색 (Yellow) | Color (RGBA) | 색상 정보를 포함하며, RGB 채널과 알파(Alpha) 값을 전달함 |
보라색 (Purple) | Vector (XYZ) | 3차원 좌표 정보를 의미하며, 위치, 회전, 법선 벡터 등을 포함 |
회색 (Grey) | Value (Scalar) | 실수(float)를 의미하며, 마스크나 강도 조절에 사용됨 |
초록색 (Green) | Shader (Closure) | 빛의 계산 방식을 담은 셰이더 데이터로, 최종 출력 노드에만 연결됨 |

## keys

- f : 그룹 묶기. 라벨링 가능 (4.2 버전 이상)
- ctrl + T : 자동 생성? 뭐지
- ctrl + G : 그룹 생성. 선택한 노드들을 하나의 노드에 집어넣어 단순화 하는 기능
    - Tab 키를 누르면 내부로 들어갈 수 있다
