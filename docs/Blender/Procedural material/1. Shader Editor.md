# 1. Shader Editor
블렌더 기초 쉐이더 에디터 알아보기

---

## 머티리얼 생성
1. 매쉬 선택된 상태에서
2. 오른쪽 패널에서 빨간 공 아이콘(Material Properties) 클릭
3. New 버튼 클릭

> 새 머티리얼이 생성되고 선택된 매쉬에 자동으로 적용됨

- 블렌더 **기본 뷰포트 모드는 머티리얼이 안 보이는 모드**
- 3D 뷰포트 오른쪽 위에 있는 구 아이콘들 중
    - Material Preview (체커무늬 구) or Rendered 클릭
    - **머티리얼 색상 보임**

---

## Shader Editor의 기본 구성
Shader Editor를 열면 항상 최소 2개의 노드가 존재

```md
[ Principled BSDF ] ───▶ [ Material Output ]
```

1. **Material Output (출력)**
    - 머티리얼의 최종 결과
2. **Principled BSDF (표준 표면 셰이더)**
    - 현실 물리 기반(PBR)

---

### 절차적 텍스쳐 생성 구조

```md
[ Noise Texture ] → [ Color / 수치 ] → [ Principled BSDF ] → [ Material Output ]
```

- **Noise Texture** = 패턴/값 생성기
- **Principled BSDF** = 표면 물리 계산기
- **Output** = 화면 출력

---

## 소켓 색상

소켓 색상 | 데이터 타입 (Data Type) | 설명 및 특징 |
|---|---|---|
노란색 (Yellow) | Color (RGBA) | 색상 정보를 포함하며, RGB 채널과 알파(Alpha) 값을 전달함 |
보라색 (Purple) | Vector (XYZ) | 3차원 좌표 정보를 의미하며, 위치, 회전, 법선 벡터 등을 포함 |
회색 (Grey) | Value (Scalar) | 실수(float)를 의미하며, 마스크나 강도 조절에 사용됨 |
초록색 (Green) | Shader (Closure) | 빛의 계산 방식을 담은 셰이더 데이터로, 최종 출력 노드에만 연결됨 |

## shortcut
단축키

- 텍스쳐 세팅 자동화, 디버깅

| 단축키 | 기능 | 조건/비고  |
|---|---|---|
| Ctrl + T | 워크 플로우 단축용 자동 세팅 매크로  | 선택한 텍스처 노드에 Mapping/Texture Coordinate 포함한 기본 세팅 자동 구성 |
| Ctrl + Shift + T | PBR 자동 세팅 | Principled BSDF에 PBR 텍스처 세트(BaseColor/Normal/Roughness 등) 선택·자동 연결 | 
| Ctrl + Shift + 노드 클릭 | 미리보기 / 디버깅 | 선택 노드의 출력을 즉시 프리뷰 (뷰어/출력에 임시 연결) |

- 링크 연결, 수정

| 단축키 | 기능 | 조건/비고 |
|---|---|---|
| Ctrl + 우클릭 드래그 | 링크 자르기 | 드래그 경로를 통과하는 모든 연결 제거 |
| Ctrl + Alt + 우클릭 드래그 | 링크 뮤트 | 연결을 임시로 비활성화 (우회) |
| Shift + 우클릭 드래그 | Reroute 추가 | 선 정리용 중간 포인트 생성 |

- 노드 삭제, 유지, 토글

| 단축키 | 기능 | 조건/비고 |
|---|---|---|
| X / Delete | 노드 삭제 | 일반 삭제 |
| Ctrl + X | 연결 유지 삭제 | 노드만 제거하고 앞뒤 연결 유지 |
| M | 노드 뮤트 | 노드를 우회 처리 (임시 비활성화) |
| H | 노드 접기 | 노드 UI 최소화 |
| Ctrl + H | 소켓 숨김 | 사용 안 하는 소켓 숨김/표시 |

- 노드 구조 정리, 주석

| 단축키 | 기능 | 조건/비고 |
|---|---|---|
| F (4.2 버전 이상) | Frame 생성 (노드 묶기) | 주석/구조용 Frame 노드 추가 |
| Ctrl + G | 노드 그룹 생성 (함수 생성) | 선택 노드를 재사용 가능한 Group으로 캡슐화 |
| Ctrl + Alt + G | 그룹 해제 | Ungroup |
| Tab | 그룹 내부 편집 | Group 안/밖 토글 |

- **f (4.2 버전 이상) / ctrl + j**
  - 주석용 프레임 생성 (노드 묶기)
- **ctrl + G** : 선택한 노드들로 **함수 생성**
    - 하나의 재사용 가능한 노드로 만들기
    - Tab 키를 누르면 내부로 들어가서 편집 가능
