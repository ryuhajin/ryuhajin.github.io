# shortcut

> 마우스 오른쪽 버튼은 언제나 '취소' 기능을 수행

## 뷰포트 조작 (Navigation)

- 마우스 휠

| 단축키 | 기능 | 설명 |
| --- | --- | --- |
| 마우스 휠 + 드래그 | 화면 회전 | 시점을 중심으로 회전 (Orbit) |
| 마우스 휠 Up/Down | 확대/축소 | 시점 줌 인/아웃 (Zoom) |
| Shift + 마우스 휠 + 드래그 | 평행 이동 | 드래그한 영역으로 화면 이동 |
| Shift + b + 원하는 영역 드래그 | 확대/축소 | 선택한 영역으로 확대 |
| alt + 마우스 휠 + 오브젝트 클릭 | 객체 집중 | 해당 오브젝트로 화면 중심 이동 |
| alt + 마우스 휠 + 원하는 뷰 각도로 드래그 | 정면 / 측면 / 평면 | 각 축의 정사영 뷰로 전환 |


- 숫자 패드

| 단축키 | 기능 | 설명 |
| --- | --- | --- |
| Numpad 0 | 카메라 뷰 | 카메라 시점 화면으로 전환 |
| Numpad 1 / 3 / 7 | 정면 / 측면 / 평면 | 각 축의 정사영 뷰로 전환 |
| Numpad 5 | 투영법 전환 | 원근법(Perspective) ↔ 등각 투영(Orthographic) |
| Numpad . (점) | 객체 집중 | 선택한 오브젝트를 화면 중앙에 꽉 차게 배치 |

- **shift + 우클릭** : 3D 커서 이동
- **shift + c** : 3D 커서 센터로 이동

- 투시 모드

| 단축키 | 기능 | 설명 |
| --- | --- | --- |
| Alt + Z | 투명 모드 | 엑스레이(X-Ray) 모드 켜기/끄기 (뒷면 선택 시 필수) |
| Shift + Z | 와이어 프레임 | 면을 숨기고 뼈대(점, 선) 만 집중해서 보기 |
| Shift + H | 선택 외 숨기기 | 선택 한 오브젝트 외 모두 숨기기 |
| Alt + H | 숨기기 취소 | 숨기기 한 오브젝트 다시 보이게 |

## 오브젝트 모드 (Object Mode)

| 단축키 | 기능 | 로직 및 팁 |
| --- | --- | --- |
| G | 이동 (Grab) | **G + X/Y/Z** 를 눌러 특정 축으로만 이동 가능 |
| R | 회전 (Rotate) | R + R 을 누르면 트랙볼 방식으로 자유 회전 |
| S | 크기 (Scale) | S + 수치 입력 시 정확한 배율 조정 가능 |
| Shift + A | 추가 (Add) | 메쉬, 조명, 카메라 등 생성 메뉴 호출 |
| X / Del | 삭제 (Delete) | 선택한 객체 삭제 (X는 확인 팝업 발생) |
| Shift + D | 복제 (Duplicate) | 복제 후 즉시 이동 모드 진입 |
| Alt + G/R/S | 변형 초기화 | 이동/회전/크기 값을 기본값(0 또는 1)으로 리셋 |

## 기타 유틸리티
- **Shift + Alt + Z** : 뷰포트 전체 오버레이 끄기

| 단축키 | 기능 | 설명 |
| --- | --- | --- |
| Z | 쉐이딩 메뉴 | 와이어프레임, 렌더 뷰 등 파이 메뉴 호출 |
| N | 사이드바 | 위치, 크기 등 속성 값이 표시되는 우측 패널 |
| T | 툴바 | 좌측 도구 상자 표시/숨김 |
| Ctrl + J | 통합 (Join) | 선택한 여러 오브젝트를 하나의 오브젝트로 합침 |
| / (Slash) | 독립 뷰 | 선택한 객체만 남기고 나머지를 일시적으로 숨김 |
| Ctrl + Z | 실행 취소 | 작업 단계 되돌리기 |

## 에디트 모드 (Edit Mode)
에디트 모드에서 1, 2, 3 키가 점/선/면 선택 모드로 작동하지 않고 화면 시점이 돌아가는 이유
- **Emulate Numpad(넘패드 에뮬레이트) 옵션**이 활성화
  - 해당 기능을 끄고 `~` (파이 메뉴) 를 눌러 시점 전환 메뉴를 사용할 수 있다
  - 혹은 자신만의 단축키 지정하기

| 단축키 | 기능 | 설명 |
| --- | --- | --- |
| Tab | 모드 전환 | 오브젝트 모드 ↔ 에디트 모드 전환 |
| 1 / 2 / 3 | 선택 모드 | 점(Vertex) / 선(Edge) / 면(Face) 선택 모드 전환 |
| E | 돌출 (Extrude) | 선택한 요소를 뽑아내어 새로운 메쉬 생성 |
| I | 면 삽입 (Inset) | 선택한 면의 안쪽에 새로운 면 삽입 |
| Y | 분리 (Split) | 선택한 메쉬를 인접한 메쉬와 연결만 끊음 (동일 오브젝트 유지) |
| P | 추출 (Separate) | 선택한 메쉬를 별도의 새로운 오브젝트로 분리 (아웃라이너에 새 객체 생성) |
| Ctrl + B | 베벨 (Bevel) | 모서리를 깎아 면을 나누고 부드럽게 처리 |
| Ctrl + R | 루프 컷 (Loop Cut) | 메쉬를 가로지르는 절단선 추가 (휠로 개수 조절) |
| K | 나이프 (Knife) | 사용자가 원하는 경로대로 직접 메쉬를 컷팅 |
| F | 면 생성 (Face) | 선택된 점이나 선들 사이에 면을 채움 |
| M | 병합 (Merge) | 선택한 점들을 하나로 합침 (중심, 거리 기준 등) |


1. **Y로 split**
  - 오브젝트 내 파편
  - 매쉬 연결만 끊어짐
  - 특정 부위만 떼어내 변형할 때 사용
2. **P로 separate**
   - 새로운 오브젝트로 생성
   - 아웃라이너에 별도 객체 생김
   - 파츠를 완전히 분리하여 관리할 때
