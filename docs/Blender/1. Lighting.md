# 1. Lighting

# Light 종류
**Z 버튼 누르면 화면 모드 휠** 나타남. Rendered 선택하면 렌더링 된 뷰 보임 

## Point
1. **Radius 반경은 그림자에 영향을 준다**
    - Radius가 0에 근접하면 mask처럼 잘리는 그림자 생성
    - Radius가 늘어날수록 그림자가 부드럽게 섞임

---

## Sun
1. **Angle은 그림자에 영향을 준다**
    - 각도가 0에 근접하면 mask처럼 잘리는 그림자 생성
    - 각도가 늘어날수록 그림자가 부드럽게 섞임

---

## Spot
1. Beam Shape의 Spot Size = 원뿔의 원 사이즈 조절 가능
2. Radius는 빛의 가장자리에 영향을 줌
    - 클수록 부드럽게 페이드 아웃됨
3. Beam shape의 Blend = 클수록 빛의 가장자리가 더 부드러워짐

### Spot의 Nodes (spot 라이트에서만 사용 가능한 방법!)
shader editor를 통해 이미지를 조명 색으로 사용 가능하다

- cycles 엔진에서만 사용 가능

### 라이트에 이미지 넣기
1. nodes 탭에 Use Nodes 버튼 클릭
2. 창을 나누고 shader editor로 들어감
3. light output 노드와 함께 이미 기본 그래프 (예: Emission - light output)가 생성되어 있음
4. image texture 노드를 추가해 open으로 사용하고 싶은 조명 이미지 추가
5. tex coordinate 노드 추가, mapping 노드 추가
6. tex coordinate (normal) - mapping - image texture - emission - light output
7. mapping으로 이미지 위치를 조절하거나 image texture의 옵션을 조절하여 원하는 이미지를 조명으로 출력 가능

---

## Area
평면과 빛 방향이 있는 라이트

1. 림라이트를 만들 때 - 평면 shape를 Rectangle로 사용
    - 가로세로 비율이 차이나는 길쭉한 직사각형으로 만들기
    - 림 라이트나 아주 얇은 빛이 추가됨
2. 만약 부드러운 림라이트를 만들고 싶으면 Elilpse로 만들기 (타원형)
3. Beam Shape Spread로 확산 값 변경
    - 확산 값을 낮추면 빛이 작아지고 한 영역만 집중적으로 비춤 (선명)
    - 확산 값을 올리면 반대 (부드러움)

> 블렌더에서는 빛도 애니메이션이 가능하다!
> - 전력 (power) / color 등을 키프레임으로 잡아서 사용 (i키로 키프레임 추가)
> - 블렌더는 거의 모든 것을 애니메이션 화 할 수 있다
