# 3. CPU bound
CPU bound를 이해하기 위한 개념 정리

---

## CPU bound란?
CPU bound는 **프레임을 만드는 전체 과정에서 CPU가 더 오래 걸려서 병목이 되는 상태**를 말함

- 즉, GPU는 아직 더 그릴 여유가 있는데, CPU가 다음 프레임 준비를 늦게 끝내서 전체 FPS가 떨어지는 상황
